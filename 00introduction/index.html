Introduction de SASS

SASS est un préprocesseur qu'on trouve dans pas mal des projets et qui a l'avantage d'integrer
tout un tas de fonctionnalité qui permet d'écrire du css plus organisé et plus propre
L'extension d'un préprocesseur SASS est en .scss(tout les css sont des scss valable) ou
.sass(qui enlève tout les ";" et les "{}")

Installation SASS

npm install -g sass

Utilisation de SASS

sass fichier.scss : voir un css correspondant au fichier.scss
sass fichier.scss fichier.css : génerer un fichier.css correspondant au fichier.scss
sass fichier.scss fichier.css --watch : regénerer un fichier.css correspondant au fichier.scss automatiquement
